# Shadowlol Web

### What is Shadow Lol Web

* This is a League Of Legends data web, in this web you can access to Stadistics Of champs such as:
  * Win rate
  * Averages
  * Builds
  * And more data

### Is the web site running at the moment?
  * No, this web is currently under development

### Other information

  * Django
  * Materialize CSS
  * Cassiopeia Riot Api
 * Is writting using:
  * Python
  * Html5
